# linktree-frontend

Apple 스타일 Linktree + Admin 대시보드 (React)

## 페이지

- `/` — 링크 메인 페이지
- `/admin` — 통계 어드민 (방문수, 클릭수, 차트)

## 링크 수정

`src/links.js` 에서 링크 목록 편집

## 환경변수

`.env.example` 참고

## 클라우드타입 배포

- 서비스 타입: Node.js (또는 Static)
- 빌드 명령: `npm install && npm run build`
- 서비스 디렉토리: `build/`
- 포트: 3000 (serve 사용 시)
